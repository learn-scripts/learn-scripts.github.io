Learn Scripts Online!

A collection of resources and tools for learning non-Latin writing systems.

Scripts currently available:
  • Hiragana (ひらがな)
  • Katakana (カタカナ)
  • Hebrew Alphabet (עִבְרִית)
  • Georgian Alphabet (მხედრული)
  • Greek Alphabet (ἀλφάβητος)
  • Glagolitic (ⰳⰾⰰⰳⱁⰾⰹⱌⰰ)
  • Russian Cyrillic (Пусский Кири́ллица)
  • Devanagari (देवनागरी)
  • Bopomofo/Zhuyin (ㄅㄆㄇㄈ)
